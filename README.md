Handwritten Character Recognition

Project Description

This project uses Machine Learning to recognize handwritten digits.
The model is trained using the MNIST dataset and can predict digits from handwritten images.

Technologies Used

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib
- Pandas

Features

- Train a model using handwritten digit dataset
- Preprocess images using OpenCV
- Predict handwritten digits
- Simple and beginner-friendly machine learning project

Dataset

This project uses the MNIST dataset, which contains 70,000 handwritten digit images (0–9).

Project Structure

handwritten-character-recognition
│
├── train_model.py
├── predict.py
├── preprocess.py
├── requirements.txt
├── digit.png
└── README.md

Installation

Install required libraries using pip:

pip install tensorflow keras opencv-python numpy matplotlib pandas

How to Run

1. Train the Model

python train_model.py

2. Predict Handwritten Digit

python predict.py

Output

The program will predict the handwritten digit from the input image.

Example Output:
Predicted Digit: 5
